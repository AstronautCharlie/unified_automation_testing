  ---
  How much time each individual spends standing, walking, sitting, and lying 
  down

  Method

  File used: ADL_no_label.parquet — 644,635 rows × 245 columns; 4 subjects
  (S1–S4), 5 ADL recordings each, sampled at 30 Hz (33.3 ms/sample).

  Two sensor-derived features:

  1. Back tilt from vertical — computed from the back-mounted IMU accelerometer
  (InertialMeasurementUnit_BACK_accX/Y/Z):

  back_tilt (°) = arctan2( sqrt(accY² + accZ²), |accX| )

  1. When upright, gravity aligns with the spine axis (accX ≈ −1000 units ≈
  −1g), giving tilt ≈ 0°. When horizontal (lying), gravity shifts to the
  perpendicular axes, giving tilt ≈ 90°. Verified: the back IMU magnitude ≈ 1000
   units at rest, confirming the coordinate geometry.
  2. Shoe angular velocity magnitude — combined from left and right shoe IMUs:

  shoe_angvel = sqrt( Σ (L-SHOE_AngVelBodyFrame_XYZ)² +
  (R-SHOE_AngVelBodyFrame_XYZ)² )

  2. Distribution: median ≈ 258 (stationary), 75th percentile ≈ 3100 (walking).
  The large gap between these confirms a bimodal pattern.

  Smoothing: Rolling median over 30 samples (1 second) per recording, to reduce
  momentary noise.

  Classification rules (priority order):

  ┌──────────┬───────────────────────────────────────────────────────────────┐
  │  State   │                             Rule                              │
  ├──────────┼───────────────────────────────────────────────────────────────┤
  │ Lying    │ smooth_tilt > 55° — back is close to horizontal               │
  ├──────────┼───────────────────────────────────────────────────────────────┤
  │ Walking  │ smooth_tilt ≤ 55° AND smooth_angvel > 2000 — upright with     │
  │          │ active feet                                                   │
  ├──────────┼───────────────────────────────────────────────────────────────┤
  │ Sitting  │ smooth_tilt > 25° (and not lying, not walking) — partial      │
  │          │ lean, feet still                                              │
  ├──────────┼───────────────────────────────────────────────────────────────┤
  │ Standing │ smooth_tilt ≤ 25° (and not lying, not walking) — upright,     │
  │          │ feet still                                                    │
  └──────────┴───────────────────────────────────────────────────────────────┘

  Thresholds were derived from the observed data distributions (the 90th
  percentile of back_tilt under low shoe movement is ~61°; the clear bimodal gap
   in shoe_angvel sits between 258 and 3101).
