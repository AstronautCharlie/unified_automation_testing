  ---
  How to Detect Locomotion Status from Sensor Data

  Preliminary: sensor units. All accelerometers output in milli-g, where resting
   gravity ≈ 1000 units (≈9.8 m/s²). At rest, the vector magnitude is always
  ~1000; what changes across postures is how gravity projects onto each sensor's
   X/Y/Z axes, which depends entirely on how the sensor is oriented relative to
  gravity.

  There are four states to distinguish: Lie → Sit → Stand → Walk. The cleanest
  approach treats this as a cascade.

  ---
  Step 1 — Detect Lying (vs. all upright states)

  Best sensor: Accelerometer_HIP_accY

  ┌────────────┬────────┬─────────┬──────────┐
  │ Locomotion │ Median │ 5th pct │ 95th pct │
  ├────────────┼────────┼─────────┼──────────┤
  │ Lie        │ 143    │ -172    │ 497      │
  ├────────────┼────────┼─────────┼──────────┤
  │ Sit        │ 983    │ 841     │ 1024     │
  ├────────────┼────────┼─────────┼──────────┤
  │ Stand      │ 937    │ 486     │ 1040     │
  ├────────────┼────────┼─────────┼──────────┤
  │ Walk       │ 946    │ 707     │ 1276     │
  └────────────┴────────┴─────────┴──────────┘

  Threshold: HIP_accY < 500 → Lying (95.2% sensitivity, <1% false positive rate
  for Sit/Walk, ~5% for Stand)

  Physical explanation: The hip accelerometer's Y-axis is oriented roughly
  perpendicular to the body's long axis. When the person is upright (sitting,
  standing, walking), gravity pulls straight down through the body, which aligns
   closely with the Y-axis — so Y carries most of the ~1000-unit gravitational
  signal. When the person lies down, the torso becomes horizontal. Now gravity
  is perpendicular to the body long axis entirely, and the Y-axis of the sensor
  ends up pointing sideways (roughly horizontal), so it receives only a small
  gravity component (~143 instead of ~950). This is essentially the sensor
  detecting which way is "down" relative to the wearer's torso.

  Supporting sensor: InertialMeasurementUnit_L-SHOE_EuY (Euler pitch angle of
  the left shoe)

  ┌────────────┬────────┐
  │ Locomotion │ Median │
  ├────────────┼────────┤
  │ Lie        │ 0°     │
  ├────────────┼────────┤
  │ Sit        │ −48°   │
  ├────────────┼────────┤
  │ Stand      │ −40°   │
  ├────────────┼────────┤
  │ Walk       │ −43°   │
  └────────────┴────────┘

  Threshold: SHOE_EuY > −20 → Lying (74% sensitivity, ~3% FP). The foot lies
  flat on the floor when the person is lying down (pitch ≈ 0°), whereas during
  all upright states the foot rests in its natural plantar-flexed position
  (~40–48° of pitch). Less sensitive alone but useful as corroboration.

  ---
  Step 2 — Detect Walking (in upright states)

  Best sensor: InertialMeasurementUnit_L-SHOE_AngVelBodyFrameX and Y (or
  equivalently, the same-named NavFrame columns — they are identical in this
  dataset)

  ┌────────────┬────────────────────────┬───────────────────┐
  │ Locomotion │   Mean std over time   │ Within-sample std │
  ├────────────┼────────────────────────┼───────────────────┤
  │ Walk       │ ~2,228 (rolling 1-sec) │ 2,378             │
  ├────────────┼────────────────────────┼───────────────────┤
  │ Stand      │ ~550 (rolling 1-sec)   │ 607               │
  ├────────────┼────────────────────────┼───────────────────┤
  │ Sit        │ ~45 (rolling 1-sec)    │ 279               │
  ├────────────┼────────────────────────┼───────────────────┤
  │ Lie        │ ~119 (rolling 1-sec)   │ 393               │
  └────────────┴────────────────────────┴───────────────────┘

  Threshold: rolling 1-second standard deviation of shoe angular velocity 
  magnitude > ~500–800 → Walking (94% sensitivity at threshold 500, ~18% FP from
   Stand; tighten to ~800 for better precision).

  Alternatively, at a single timestep: instantaneous angular velocity magnitude
  > 1500 → likely mid-step (47% of Walk frames, <11% of Stand frames, <5% of
  Sit/Lie).

  Physical explanation: During walking, the foot goes through a dramatic
  rotation cycle with every step: it lifts, swings forward, and strikes the
  ground. The angular velocity sensors on the shoe detect this rotation — the
  foot pitch rate is extremely high during toe-off and heel-strike phases.
  Between steps the signal returns to near-zero briefly, creating a periodic
  high-amplitude oscillation. The standard deviation of angular velocity over
  roughly one gait cycle (~1 second at ~1 step/sec) is therefore very high
  (~2200) for walking and remains low for standing (~550, which captures only
  postural sway) and very low for sitting (~45) and lying (~120). The Nav-frame
  linear acceleration L-SHOE_Nav_Ax/Ay also works well (Walk std ~630 vs Stand
  ~213 vs Sit ~80) — this captures the heel-strike impact and push-off dynamics.

  ---
  Step 3 — Distinguish Sitting from Standing (static upright states)

  Best sensor: Accelerometer_RKN^_accZ (right knee, Z-axis — the sensor labeled
  RKN^)

  ┌────────────┬────────┬──────────┬──────────┐
  │ Locomotion │ Median │ 10th pct │ 90th pct │
  ├────────────┼────────┼──────────┼──────────┤
  │ Sit        │ 952    │ 888      │ 1009     │
  ├────────────┼────────┼──────────┼──────────┤
  │ Stand      │ 342    │ −69      │ 649      │
  ├────────────┼────────┼──────────┼──────────┤
  │ Walk       │ 218    │ −231     │ 819      │
  ├────────────┼────────┼──────────┼──────────┤
  │ Lie        │ 224    │ −323     │ 962      │
  └────────────┴────────┴──────────┴──────────┘

  Threshold: RKN^_accZ > 600 → Sitting (97.7% sensitivity, ~14% FP from
  Stand/Walk)

  Physical explanation: When standing, the leg is roughly vertical. Gravity
  (~1000 units) falls straight down, which for the knee sensor means it loads
  primarily onto the Y-axis (along the leg's long axis), giving accY ~944. The
  Z-axis, which points sideways across the knee, receives very little
  gravitational force (accZ ~342). When sitting, the knee bends to approximately
   90°. This rotation shifts which axis gravity aligns with: now gravity
  projects strongly onto the Z-axis of the knee sensor (accZ ~952), while Y
  drops dramatically (accY ~347). You can think of it as: whichever axis of the
  knee sensor has high gravity load tells you whether the knee is straight (load
   on Y = standing) or bent (load on Z = sitting).

  Supporting sensor: Accelerometer_BACK_accZ

  ┌────────────┬────────┐
  │ Locomotion │ Median │
  ├────────────┼────────┤
  │ Lie        │ −30    │
  ├────────────┼────────┤
  │ Sit        │ −382   │
  ├────────────┼────────┤
  │ Stand      │ −590   │
  ├────────────┼────────┤
  │ Walk       │ −446   │
  └────────────┴────────┘

  Threshold: BACK_accZ < −500 → Standing (61% sensitivity, 35% FP from Sit).
  When standing, the back is most vertical, so gravity projects most strongly
  onto the Z-axis of the back sensor (which points posterior-to-anterior when
  the back is upright). When sitting, the person may recline slightly or the
  hip-back angle changes, reducing the Z projection. Combining knee Z and back Z
   gives better discrimination than either alone.

  ---
  Summary Decision Logic

  IF HIP_accY < 500 → LIE
  ELSE IF rolling_std(SHOE_AngVelMagnitude, 1sec) > 700 → WALK
  ELSE IF RKN^_accZ > 600 → SIT
  ELSE → STAND

  ┌────────────┬───────────────┐
  │ Locomotion │ Median Z (mm) │
  ├────────────┼───────────────┤
  │ Lie        │ ~486          │
  ├────────────┼───────────────┤

  These are physically sensible: lying person's chest is ~0.5m off the floor,
  seated ~1.1m, standing ~1.4m. Stand and Walk are nearly identical in height
  (as expected), confirming that the tag height alone cannot separate them —
  that's where the shoe angular velocity becomes essential.

  ---
  Which Sensors Matter Most (Ranked by Discriminative Power)

  1. Accelerometer_HIP_accY — best single sensor for Lie vs upright
  2. Accelerometer_RKN^_accY and RKN^_accZ together — best for Sit vs Stand
  3. InertialMeasurementUnit_L-SHOE_AngVelBodyFrameX/Y — best for Walk detection

  4. Accelerometer_BACK_accZ / IMU_BACK_accZ — secondary confirmation of Stand
  vs Sit

  6. InertialMeasurementUnit_LLA_accX (left lower arm) — useful corroborator;
  Lie/Sit: ~+200, Stand/Walk: ~−800, because the arm hangs vertically in both
  standing and walking, flipping the axis that gravity loads
