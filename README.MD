This repository contains my attempt at getting Claude Code to detect patterns in the UCI Opportunity dataset. 

Outputs of initial experiments `prompt_<x>` folders, where each one represents a different prompt given to Claude Code.

See `notes.MD` for description of experiments and results.

See `utils.py` for useful constants and functions.

Now I'm moving on to building an automated pipeline to use the documentation in place of my prompts (see `pipeline` folder)
